Sample Static Grunt Project

A sample project that compiles less and coffeescript files as well as combining static files for release to production. This builds purely static flat files and is an easy way to make it nicer to build a static HTML page.

Work in the src directory. To auto-compile less and coffeescript files, run: 

grunt watch

Compile the project with: 

grunt

For more details for the README.md file in this project.
